# Moon Configuration

Moonrepo workspace configuration for task orchestration.

## Task Inheritance

Projects inherit tasks from `tasks/tag-*.yml` based on their `tags` in `moon.yml`. Available tags: `cli`, `go`, `npm`, `shell`, `skill`, `command`, `tsconfig`, `typescript-config`, `typescript-script`, `typescript`.
