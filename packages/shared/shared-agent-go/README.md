# Shared Agent Go

Shared Go library for agent CLI functionality, used by [agent-cli-go](../../agent/agent-cli-go/).

## Packages

| Package         | Description                             |
| --------------- | --------------------------------------- |
| `pkg/agents`    | Agent type definitions and runner logic |
| `pkg/config`    | Configuration loading and validation    |
| `pkg/providers` | Model provider implementations          |
| `pkg/types`     | Shared type definitions                 |
| `pkg/worktree`  | Git worktree management                 |
