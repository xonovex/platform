# @xonovex/agent-cli-go-win32-x64

Windows x64 binary for `@xonovex/agent-cli-go`.

## Overview

This package contains the pre-built Windows x64 binary for the agent-cli-go tool. It is automatically installed as an optional dependency when you install `@xonovex/agent-cli-go` on a compatible platform.

## Direct Installation

```bash
npm install @xonovex/agent-cli-go-win32-x64
```

## Platform Requirements

- **OS**: Windows
- **Architecture**: x64

## Usage

This package is not meant to be used directly. Install `@xonovex/agent-cli-go` instead, which will automatically select the correct platform binary.
