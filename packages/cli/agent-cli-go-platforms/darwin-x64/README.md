# @xonovex/agent-cli-go-darwin-x64

macOS x64 binary for `@xonovex/agent-cli-go`.

## Overview

This package contains the pre-built macOS x64 (Intel) binary for the agent-cli-go tool. It is automatically installed as an optional dependency when you install `@xonovex/agent-cli-go` on a compatible platform.

## Direct Installation

```bash
npm install @xonovex/agent-cli-go-darwin-x64
```

## Platform Requirements

- **OS**: macOS
- **Architecture**: x64 (Intel)

## Usage

This package is not meant to be used directly. Install `@xonovex/agent-cli-go` instead, which will automatically select the correct platform binary.
